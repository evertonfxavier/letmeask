## 🚀 Getting Started 

Clone the project and access the folder.
```bash
#Clone project
$ git clone https://github.com/AmandaCarvalho1989/go-restaurant.git
```

Follow the steps below
```
# install the dependecies 
$ npm install or yarn  
 
# Start projet
$ npm start or yarn run start
```
